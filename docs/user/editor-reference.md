# エディタリファレンス

## メインタブ

| ショートカット | タブ | 主な機能 |
|---------------|------|----------|
| Ctrl+1 | プロジェクト | 新規・開く・保存・書き出し・章・プラグイン・支援・ストレージ設定 |
| Ctrl+2 | データベース | ユニット / クラス / 武器 / アイテム / スキル / 地形 |
| Ctrl+3 | マップ | 地形描画・ユニット配置・勝敗条件 |
| Ctrl+4 | テストプレイ | ランタイムを iframe で起動・シード・無敵デバッグ |
| Ctrl+5 | イベント | トリガー別コマンド列の編集 |
| Ctrl+6 | 素材 | 画像・BGM・SE のインポート一覧 |

ヘッダーにプロジェクト名と未保存マーク（`*`）が表示されます。

## プロジェクトタブ

| 操作 | 説明 |
|------|------|
| 新規（サンプル） | 同梱サンプルを読み込み |
| 開く | JSON ファイルまたはフォルダ（デスクトップ版） |
| 保存 | 現在の形式で上書き |
| テンプレート再読込 | サンプル定義のみ再取得（要確認） |
| HTML5 / Electron / モバイル書き出し | zip をダウンロード |
| 保存バックエンド | ローカル（既定）／クラウド（準備中スタブ） |

章パネルで章名の編集・マップ紐付け。プラグインの有効/無効、支援会話一覧もここで確認できます。

## データベースタブ

左リストでエントリを選択し、右フォームで編集します。変更は即座にプロジェクト状態に反映され、**未保存** になります。

## マップタブ

| ツール | 操作 |
|--------|------|
| ペン | 地形タイルの塗りつぶし |
| ユニット | セルをクリックで配置（Shift で連続） |
| 陣営 | プレイヤー / 敵 / 第三勢力 |
| 消しゴム | 配置・地形の削除 |

マップ選択、勝利条件（敵全滅・ボス撃破・ターン生存・拠点防衛）のドロップダウンがあります。

## イベントタブ

イベント ID ごとにコマンド行を追加。コマンド種別は [event-commands.md](../event-commands.md) の 15 種に限定（MVP 凍結）。

## 素材タブ

| ボタン | 追加先 |
|--------|--------|
| 画像を追加 | `assets/images/` |
| BGMを追加 | `assets/audio/bgm/` |
| SEを追加 | `assets/audio/se/` |

一覧から削除可能。件数はプロジェクトタブの **素材ファイル** に表示されます。

## テストプレイタブ

- **テストプレイ開始** — 埋め込みランタイムで現在プロジェクトを実行
- **デバッグシード** — 戦闘乱数の固定（再現テスト用）
- **無敵** — プレイヤーユニットがダメージを受けない（デバッグ）

キャンペーン（章あり）プロジェクトでは、タイトル → 拠点 → 出撃の順で戦闘に入ります。

## アクセシビリティ

キーボードのみでタブ切替・フォーム操作が可能です（spec 6）。ゲーム側はキーボード・タッチに対応（ゲームパッドはランタイムの今後拡張）。
