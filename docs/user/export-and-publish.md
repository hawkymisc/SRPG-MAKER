# 書き出しと配布

完成したプロジェクトを、プレイヤー向けの成果物として配布します。

## 書き出しの種類

プロジェクトタブから zip をダウンロードします。

| 種類 | 内容 | 配布先の目安 |
|------|------|--------------|
| **HTML5** | `game/` フォルダ + 静的ファイル | Web サーバ、itch.io、GitHub Pages |
| **Electron** | HTML5 ゲーム + デスクトップシェル | Windows / macOS / Linux 向け zip |
| **モバイル** | Capacitor 用プロジェクト骨格 | Android Studio / Xcode で実機ビルド |

## HTML5（最も手軽）

1. プロジェクトタブ → **HTML5書き出し**
2. ダウンロードした zip を展開
3. `game/` 以下を Web サーバで公開（`file://` 直開きは環境によって制限あり）

ローカル確認例:

```bash
cd game
npx serve .
# または python -m http.server 8080
```

## Electron（デスクトップ）

1. **Electron書き出し** で zip を取得
2. 展開後、README の手順に従い各 OS 向けにビルド（配布形態は ADR-0004 / 0005 参照）
3. 作成者が配布するのはビルド済み実行ファイルまたはストア申請用パッケージ

開発者向けのビルド詳細は [contributor/setup.md](../contributor/setup.md) および `packages/editor-electron/README.md` を参照してください。

## モバイル（Capacitor）

1. **モバイル書き出し** で zip を取得
2. Android Studio / Xcode で Capacitor プロジェクトを開き実機ビルド

実機での音声・タッチ操作は別途確認が必要です（ADR-0007）。

## 書き出しに含まれるもの

- ランタイム（Phaser 3 ビルド）
- プロジェクト JSON（分割マージ済み）
- `assets/` に登録した素材
- 有効化したプラグイン

## 配布前チェックリスト

- [ ] 書き出し zip を別 PC またはシークレットウィンドウで起動
- [ ] タイトル → 最終章クリア（または意図した終了）まで通しプレイ
- [ ] 素材のクレジット・ライセンス表記（タイトル画面や README に記載）
- [ ] セーブデータの保存先（ブラウザの LocalStorage 等）をプレイヤーに案内する場合は説明文を用意

## バージョン互換

古いツールで書き出した成果物を新しいランタイムだけで動かす場合、`schemaVersion` の互換に注意してください。原則として **書き出し時のツール版と同じ世代のランタイム** が同梱されます。

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| zip が空 / ダウンロード失敗 | ブラウザのポップアップブロックを確認。デスクトップ版を試す |
| 真っ黒画面 | Web サーバ経由で配信しているか確認。コンソールのエラーを確認 |
| BGM が鳴らない | ブラウザの自動再生ポリシー。一度画面クリック後に再生されるか確認 |
| 素材が欠ける | 素材タブに登録済みか、書き出し前に保存したか確認 |
