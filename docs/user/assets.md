# 素材

画像・BGM・SE をプロジェクトに追加し、書き出し zip に同梱できます。

## 追加方法

**素材タブ（Ctrl+6）** からファイルを選びます。

| ボタン | 保存先 |
|--------|--------|
| 画像を追加 | `assets/images/` |
| BGMを追加 | `assets/audio/bgm/` |
| SEを追加 | `assets/audio/se/` |

一覧から不要な素材を削除できます。プロジェクトタブの **素材ファイル** に件数が表示されます。

## 推奨形式

| 種別 | 形式 | 目安 |
|------|------|------|
| タイル・ユニット画像 | PNG | 透過あり可。タイルサイズはプロジェクト設定（32/48/64px）に合わせる |
| 顔グラ | PNG | 会話ウィンドウ用 |
| BGM / SE | OGG または MP3 | ブラウザ互換を優先するなら OGG |

## イベント・DB との紐付け

- 会話の顔グラ — `SHOW_MESSAGE` の `faceId`
- BGM — `PLAY_BGM` の `bgmId`（ファイル名ベースの ID）
- SE — `PLAY_SE` の `seId`

ID は素材タブに表示される名前と一致させてください。

## 同梱サンプル

`templates/sample/assets/` に CC0 等のサンプルがあります。利用条件は同フォルダの README を参照してください。

## 書き出し時の扱い

HTML5 / Electron / モバイルいずれの書き出しでも、プロジェクトに登録された `assets/` が zip に含まれます。大容量素材は配布サイズに直結するため、必要なファイルだけ登録することを推奨します。

## フォルダ保存時

デスクトップ版でフォルダ形式保存している場合、素材は `MyGame/assets/` 以下に実ファイルとして置かれます。外部ツールで画像を差し替えたあと、エディタで **再読み込み** または再起動してください。
