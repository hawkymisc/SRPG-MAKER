# Project assets (spec.md section 7)

Place binary assets here:

- `images/` — tilesets, unit sprites, faces, UI
- `audio/bgm/` — background music (`PLAY_BGM` bgmId)
- `audio/se/` — sound effects (`PLAY_SE` seId)

Filenames should match event command IDs (e.g. `bgm_intro.ogg`, `se_hit.mp3`).

JSON data stays outside `assets/` (database/, maps/, events/, etc.).
