# M4委任プロンプト:イベントシステムとHTML5書き出し(MVP完成)

前提:M3完了。

---

マイルストーンM4を実施してください。ゴールは「イベント付きの章を作って、zipひとつで配布できる状態(MVPβ)」です。

## タスク

1. **イベントコマンドのスキーマ**(schema-architectに委任)
   `docs/event-commands.md` に確定済みのコマンド15種とトリガー7種を、同ファイルのパラメータ定義どおりにZodで定義する。リストにないコマンドの追加は禁止(凍結済み)。

2. **イベントインタプリタ**(battle-logic-devとruntime-devで分担)
   条件評価・変数操作など純粋な部分はshared、メッセージ表示・演出はruntimeのasync generatorインタプリタ。各コマンドに単体テスト。

3. **イベントエディタ**(editor-devに委任)
   コマンドリスト方式のUI(追加/並べ替え/インデント表示の条件分岐)。トリガー設定UI。

4. **HTML5書き出し**(editor-devに委任)
   プロジェクト+ランタイムを単一zipにバンドルする書き出し機能。index.htmlを開くだけで動作すること。

5. **書き出し成果物のE2E**(test-engineerに委任)
   書き出したzipを展開し、Playwrightで起動 → イベント発火 → クリアまで検証するテスト。

6. **MVP受け入れテスト**(オーケストレーターが実施)
   仕様書2.1のMVP項目を1つずつ照合し、未達項目を列挙して人間に報告する。

## 完了条件
- 書き出し成果物E2Eを含む全テストグリーン、MVP項目の照合報告完了
