# M3委任プロンプト:エディタ

前提:M2完了。担当は主にeditor-dev。

---

マイルストーンM3を実施してください。ゴールは「GUIだけで章を1つ作り、テストプレイでクリアできる状態」です。

## タスク

1. **スキーマ駆動フォーム基盤**(editor-devに委任)
   Zodスキーマからフォームを自動生成する基盤(`schema-form/`)。文字列/数値/列挙/ID参照(他テーブルへのセレクタ)/配列に対応。これを使ってDBエディタの全タブ(ユニット/クラス/武器/アイテム/スキル/地形)を構築。

2. **プロジェクト管理**(editor-devに委任)
   新規作成(テンプレート選択)/開く/保存(スキーマ検証+アトミック書込)/自動バックアップ5世代。ブラウザ版はFile System Access API、将来のElectron版と抽象化層で分離。

3. **マップエディタ**(editor-devに委任)
   タイルパレット、ペン/矩形/塗りつぶし、レイヤー3層、Undo/Redo、ユニット配置(所属/AIタイプ/増援ターン)、勝敗条件設定UI。

4. **テストプレイ統合**(editor-devに委任)
   編集中の章をワンクリックでruntimeに渡して起動。デバッグパネル(無敵/変数ビューア/シード固定再生)。

5. **一気通貫E2E**(test-engineerに委任)
   Playwrightで「新規プロジェクト → マップ作成 → 敵1体配置 → テストプレイ → 勝利画面」を自動実行するシナリオを作成し、CIに組み込む。

## 完了条件
- 一気通貫E2Eグリーン、キーボードのみで主要操作が可能(E2Eで検証)
